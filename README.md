`gorecurcopy` copies directories recursively without external dependencies. Compatible with OSX, Linux, and Windows.

Example:

```go
import (
  "github.com/plus3it/gorecurcopy"
)

...

err := gorecurcopy.CopyDirectory("directory", "new_directory")
```
